Tag Html
ลำดับที่ tag ความหมาย รูปแบบการใส่ข้อมูล
1 "< html >" เป็นแท็กเริ่มต้นและสิ้นสุดของเอกสาร HTML < html >…< /html >
2 < head > เป็นแท็กที่ใช้กำหนดในส่วนที่เป็นชื่อเรื่อง < head >…< /head >
3 < title > กำหนดข้อความบนไตเติลบาร์เป็นคำสั่งในส่วน< head > < head>< title>ข้อความ< /title>< /head>
4 < meta > เป็นแท็กสำหรับแสดงข่าวสารของไฟล์เอกสาร HTML < meta>
5 < body > เป็นแท็กเริ่มต้นและสิ้นสุดของเนื้อหาในเอกสาร ซึ่งในแท็ก< body> จะมีแท็กอื่น ๆ แทรกอยู่ระหว่างแท็กเปิดและแท็กปิด < body>…< /body>
6 < h1 > เป็นแท็กสำหรับกำหนดหัวข้อให้เป็นตัวอักษรขนาดใหญ่สุดถึงเล็กสุดกำหนดให้ไม่เกิน 6 < h1>…< /h1>
7 < br> ขึ้นบรรทัดใหม่ < br>
8 < hr> ขีดเส้นคั้น < hr>
9 < background> ใช้รวมกับ< body>หรือ< table>เพื่อกำหนดสีของพื้นหลังเป็นรูปภาพ < body background= "ชื่อ.นามสกุลรูปภาพ">< /body>
10 < bgcolor> ใช้รวมกับ< body>หรือ< table>เพื่อกำหนดสีของพื้นหลัง < body bgcolor=ชื่อสี>< /body>
11 < b> ข้อความหนา < b>ข้อความ< /b>
12 < i> ข้อความเอียง < i>ข้อความ< /i>
13 < u> ข้อความขีดเส้นใต้ < u>ข้อความ< /u>
14 < font color, size, face>< /font> แสดงข้อความกำหนดสีของข้อความ กำหนดขนาดของข้อความกำหนดชนิดของข้อความ < font color=ชื่อสี size= "ขนาดกำหนดได้ตั้งแต่ 1 ถึง 7" face= "ชนิดของตัวอักษรเช่น Angsana new เป็นต้น">ข้อความ< /font>
15 < marquee> < marquee behavior=alternat>< /marquee> ทำให้ข้อความเลื่อนจากซ้ายไปขวาทำให้ข้อความเลื่อนจากซ้ายไปขวาแล้วย้อนกลับ < marquee>ข้อความ< /marquee> < marquee behavior=alternat>ข้อความ< /marquee>
16 < img src alt width height> แสดงรูปภาพ กำหนดรูปภาพ กำหนดข้อความเมื่อเอาเมาส์วางบนภาพ กำหนดความกว้างของภาพ กำหนดความยาวของภาพ < img src= "ชื่อ.นามสกุล" width= "ความกว้าง" height "ความยาว" alt= "ข้อความ">เช่น < img src = "lan.jpg" width= "80" height "250" alt= "ภาพโรงเรียน">
17 < tt> เป็นแท็กสำหรับกำหนดตัวอักษรเป็นตัวพิมพ์ดีด < tt>…< /tt>
18 < ul> เเป็นแท็กสำหรับแสดงรายการโดยไม่ต้องเรียงลำดับ < ul>…< /ul>
19 < ol> เป็นแท็กสำหรับแสดงรายการโดยเรียงลำดับ < ol>…< /ol>
20 < li> เป็นแท็กแสดงข้อความในแต่ละบรรทัดใน แท็ก < ul>และ < ol> < li>…< /li>
21 < a> เป็นแท็กที่กำหนดการเชื่อมโยง < a>…< /a>
22 < img> เป็นแท็กที่กำหนดการเชื่อมโยงด้วยรูปภาพ < img>…< /img>
23 < a href target> กำหนดการเชื่อมโยง กำหนดจุดเชื่อมโยง กำหนดการเปิดหน้าต่างใหม่ < a href= "จุดเชื่อมโยง" target= "ลักษณะการเปิดหน้าต่าง" >ข้อความ< /a> เช่น < a href= "http://suwitguide.com" target= "_blank">เว็บเพจของสุวิทย์< /a>
24 < table> เป็นแท็กสำหรับสร้างตาราง < table>…< /table>
25 < table width, bgcolor, border, bordercolor> สร้างตารางกำหนดความกว้างของตาราง < table width = "80% bgcolor=blue border= "2" bordercolor=red>
26 < tr> เป็นแท็กสำหรับการกำหนดแถวในตาราง < tr>…< /tr>
27 < td> เป็นแท็กสำหรับกำหนดคอลัมน์ในแถว < td>…< /td>
28 rowspan รวมแถว < td rowspan= "จำนวน แถว ที่จะรวม" >
29 colspan รวมสดมน์ < td colspan= "จำนวน cell ที่จะรวม" >
30 < caption> เป็นแท็กสำหรับกำหนดคำอธิบายในตาราง < caption>…< /caption>
31 < th> เป็นแท็กสำหรับกำหนดข้อความหัวข้อเรื่องของตาราง < th>…< /th>
32 < form> เป็นแท็กสำหรับกำหนดสร้างform < form>…< /form>
33 < input> เป็นแท็กสำหรับกำหนด inputใน form < input>…< /input>
34 < frame> เป็นแท็กสำหรับสร้าง frame < frame>…< /frame>
